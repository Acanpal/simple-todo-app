- hookのうち、UseState()とUseReducer()は状態管理
- UseEffect()は処理

## 用語
### SPA(CSR)
- Single Page Application
- ページ遷移をサーバーにリクエストせず、JavaScriptで画面を書き換えることで実現するWebアプリケーション
- 本当にhtmlは一つしかない、あくまでJSで画面を書き換えているだけ

### SSR
- Server Side Rendering
- サーバー側でHTMLを生成して返す
- Next.jsで実現可能

### React Router
- SPAだとしても、URLを中心に設計するためのライブラリ
- <Route path="/completed" element={<CompletedPage />} /> でcompletedページを定義する

### コンテナ / プレゼンテーション分離
-設計パターンの一種(少し古い)
- コンテナコンポーネント: 状態管理やビジネスロジック、API通信を担当
- プレゼンテーションコンポーネント: UIの表示を担当(再利用しやすい!)
  →便利な時もあるが、hook, dumpをうまく使ってコンテナコンポーネントで処理するのが一般的

### dumb（ダム）コンポーネント
- プレゼンテーションコンポーネントの一種
- 状態管理やビジネスロジックを持たない
- プロパティのみで動作する
- 超再利用しやすい
